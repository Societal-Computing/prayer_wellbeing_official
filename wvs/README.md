# World Values Survey (WVS) analysis

The WVS arm of the paper — the difference-in-differences–style interaction between sacred-day interview
timing and individual prayer propensity, plus the balance checks, robustness specifications, and the
analysis restricted to the Instagram countries — is implemented in **Stata**.

## Data

We use the **WVS Trends 3.0** harmonized file. We are not permitted to redistribute it. Download it from
<https://www.worldvaluessurvey.org/> (Data & Documentation → Trend files) and place it here, then update
the data path at the top of the do-file.
